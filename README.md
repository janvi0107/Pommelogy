# Pommelogy — Apple Variety Identification App

A full-stack web application that identifies and classifies apple varieties 
using image recognition and machine learning.

## 🛠 Tech Stack

**Frontend**
- React.js (component-based architecture)
- CSS Grid & Flexbox (mobile-first responsive design)
- Firebase Authentication

**Backend**
- Django REST Framework (Python)
- REST API endpoints for image retrieval, search & classification
- Firebase Firestore (real-time database)

## ✨ Features
- Search suggestions with real-time filtering
- Interactive image display and apple variety classification
- Secure user authentication via Firebase
- Fully responsive across mobile, tablet, and desktop
- Cross-browser compatible

## 📁 Project Structure

Pommelogy/

├── pommelogy-frontend/    # React.js frontend

├── AppleVariety/          # Django REST Framework backend

└── README.md


## 🚀 Getting Started

**Frontend**
```bash
cd pommelogy-frontend
npm install
npm start
```

**Backend**
```bash
cd AppleVariety
pip install -r requirements.txt
python manage.py runserver
```
